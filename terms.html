Terms of Service

CypherLock Password Generator provides a random string of characters based on onboard algorithm and user input. The resulting string of characters may be used as a personal password if the user decides to do so.

User may influence the character string length and complexity by determining the character string length and adding or removing levels of character variety within the resulting string by clicking on or off the built-in functions to add or remove upper case characters, numbers and special characters to the resulting string of characters.

User may decide to use the resulting string of characters as a personal password in order to secure user's personal, financial and/or professional information, or as a means to authentication, authorization and identity verification.

User may generate as many strings of random characters as desired, free of charge.

User may submit any password to the CypherLock Password Strength Tester to determine the level of password strength.
CypherLock Password Strength Tester will determine your password's strength based on criterias of: password length, presence of repeating characters, presence of upper case and lower case letter characters, presence of numbers and special characters in your password. Please note that the resulting password strength level should be considered merely as an indicator of your password's strength, not as a guarantee of security of the personal, private, financial or business information you decide to secure using the password you have submited for strength testing, regardless if the password you have submitted was or wasn't provided to you by CypherLock Password Generator.
For more information, please read our Disclaimer.

User may submit as many passwords to CypherLock Password Strength Tester for password strength testing as desired, free of charge.
